Count Digits in a Number

View Code Auther Details

Problem Statement:

Write a code in any programming language that counts the digits in a number.

Sample Input:

Number = 93423

Sample Output:

Total digits in number = 5

Code in c++

Press Ctrl + C to copy code (Windows).
Press Cmd + C to copy code (Mac OS).

            #include <iostream>
            using namespace std; 

            int main(){ 
                int Number;  
                cin>>Number;
                int totalDigits  = 0; 

                while(Number>0){ 
                   totalDigits = totalDigits+1; 
                   Number/=10; 
                } 

                cout<<"Total digits in number = "<< totalDigits; 
                 return 0;
            }

algorithm

After each iteration the number gets reduce by 10, means if the number initially of 105, but after iteration it become 0;

Complexity

Time Complexity: O(N) where N is digits present in number

O(Log2N) Solution

Press Ctrl + C to copy code (Windows).
Press Cmd + C to copy code (Mac OS).

            #include <iostream>
            #include <math.h>
            using namespace std; 

            int main(){ 
                int Number;  
                cin>>Number;
                int totalDigits  = log10(Number); 

                cout<<"Total digits in number = "<< totalDigits; 
                 return 0;
            }
